# 🎵 Spotify Data Analysis

Análise de dados do Spotify com Python puro — listas, dicionários e conjuntos para manipulação de dados, com visualizações interativas via Plotly.

> 🚧 Projeto em desenvolvimento — trabalho acadêmico da disciplina de Algoritmos e Estruturas de Dados.

---

## 🛠️ Tecnologias

- Python 3
- `csv` e `os` — bibliotecas padrão
- [Plotly](https://plotly.com/python/) — gráficos interativos

---

## ▶️ Como executar

```bash
# 1. Clone o repositório
git clone https://github.com/leonardorosler/spotify-data-analysis.git

# 2. Instale a dependência
pip install plotly

# 3. Execute
python spotify_analise.py
```

> O arquivo `spotify_data clean.csv` deve estar na mesma pasta que o script.
